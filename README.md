# 📰 ไม่ต้องเสียเวลาไล่ไถหน้าจอหาข่าวทั้งวัน ผู้ช่วยดึงข่าวที่คุณสนใจจากทุกที่ มาสรุปส่งเข้าอีเมลล์ทุกวันตอน 7 โมงเช้า

**ไม่มีเวลาอ่านข่าวยาวๆ ใช่ไหม?** โปรเจกต์นี้คือผู้ช่วยส่วนตัวที่จะดึงข่าวจากเว็บโปรดของคุณ ส่งให้ AI สรุปเนื้อหาเน้นๆ ให้เหลือแค่ 2-3 บรรทัด แล้วส่งตรงเข้าอีเมลทุกเช้า ช่วยให้อัปเดตทุกสถานการณ์จบได้ใน 3 นาที!

## ✨ จุดเด่น
* **ประหยัดเวลา:** สรุปใจความสำคัญจากบทความยาวๆ ให้สั้นกระชับ
* **อัตโนมัติ 100%:** ตั้งค่าครั้งเดียว ระบบทำงานเองทุกวันโดยไม่ต้องเปิดคอมพิวเตอร์
* **ใช้ฟรี:** พัฒนาด้วยเครื่องมือฟรีทั้งหมด (GitHub Actions + Google Gemini AI)

## 🚀 วิธีตั้งค่าและใช้งาน (ทำเพียงครั้งเดียว)

ไม่ต้องเขียนโค้ดเป็นก็ใช้งานได้ แค่ใส่ข้อมูลส่วนตัวลงใน **Settings > Secrets and variables > Actions** ของ GitHub จำนวน 5 ค่า ดังนี้:

| ชื่อตัวแปร (Secret) | สิ่งที่ต้องใส่ |
| :--- | :--- |
| `RSS_URL` | ลิงก์ RSS ของเว็บข่าว (เช่น `https://techcrunch.com/feed/`) |
| `GEMINI_API_KEY` | คีย์ AI ฟรีจาก [Google AI Studio](https://aistudio.google.com/) |
| `SENDER_EMAIL` | อีเมล Gmail ของคุณ (ผู้ส่ง) |
| `RECEIVER_EMAIL` | อีเมลปลายทาง (ผู้รับ) |
| `APP_PASSWORD` | รหัสผ่านแอป 16 หลักจาก [Google Security](https://myaccount.google.com/security) |

**เริ่มใช้งาน:** ไปที่แท็บ **Actions** > เลือก **Daily AI News Summarizer** > กดปุ่ม **Run workflow** (หรือรอให้ระบบรันเองตามเวลาที่ตั้งไว้) เช็กอีเมลได้เลย!
